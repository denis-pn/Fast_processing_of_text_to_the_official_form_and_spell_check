Программное обеспечение для решения задачи кейса № 5 (УльтраДжуны)

Програмное обеспечение работает в виде системного оверлея, при запуске испольнтельного 
файла появится окно с кнопкой обработки текста. 
Скопируйте в обмен буфера текст, нажмите на кнопку обработки текста 
В результате откроется окно, в которым те слова, которые скрипт изменил будут помечены красным цветом. 
Вы можете подтвердить изменения, тогда измененный текст переместится в ваш буфер обмена, или изменить род

Демонстрация работы

https://youtu.be/mYpgK5-B2Uw

Необходимые приложения и дополнения:

Работоспособность тестированась на версии Python 3.6

Библиотеки:

pyperclip  ( pip3 install pyperclip ) Чтение буфера обмена при нажатии на кнопку 

pymorphy2  ( pip install pymorphy2
pip install pymorphy2-dicts
pip install DAWG-Python )    Морфологический разбор слов

pyaspeller  ( pip install pyaspeller ) Проверка орфографии 

tkinter ( Предустановлена в Python)  Графическая оболочка
